- Ultra-connected Event Station
In the last year of my Bachelor, I am so happy we had the oopportunity to do a project like this for a course. We decided to create an ultra-connected event station, which is a smart device that can be used in events to provide various functionalities such as light warning, sound data logging, environmental monitoring, and more.

Regarding hardware, the project involved working with Raspberry Pi, ESP32, and many different types of sensors and actuators. We even designed a custom PCB with KiCad to integrate all the components together. Also, we came out with a very modular and compact prototype that could be easily deployed to a comercial solution.

On the other hand, the firmware was developed using C++. The most fascinating functionality was Bluetooth Mesh. Each ESP32 node sent packets to the closest node and this one repeated them to its neighbor to finally get to the central hub. We even modified the Bluetooth Mesh protocol to work in a different frequency band. This was all done thanks to the Radiohead library.

Finally, we developed a web application to control the device and visualize the data collected by the sensors in the main node, the Raspberry Pi. This project was a great way to learn about IoT, embedded systems, and web development, and it sparked my interest in further exploring these areas.

- Master thesis EPFL: BLE network for precise time synchronization
For my master thesis, I am currently working on a project related to IoT and healthcare in ESL lab, EPFL. It is based on VersaSens platform (link here: https://www.epfl.ch/labs/esl/research/smart-wearables/versasens/), a versatile and modular wearable system tailored for biomedical applications.

My role consists in creating a star-shaped BLE network with VersaSens as peripheral nodes, connected to a central hub that collects data from them while enabling real-time monitoring. The main challenge of the project is to achieve precise time synchronization of less than 10 microseconds, which is crucial for accurate data analysis and monitoring in many healthcare applications.

For this reason, I have to work with special timing protocols and optimize the BLE communication to minimize latency and ensure reliable data transmission. This project is a great opportunity to apply my knowledge in IoT, and I see it as a significant opportunity to work in one of the best universities of the world.
